INDIVIDUAL MARKDOWN DOCUMENTATION FILES

All documentation belongs in /docs.

3.1 — Games

Cursor must create:

docs/GAME_SCRAMBLE.md

docs/GAME_DICEFURY.md

docs/GAME_EMOJIHUNT.md

docs/GAME_SPELLING.md

docs/GAME_RPG.md

Each file must include:

Description of game

How to play

Full command list (using dashed prefixes)

Admin commands

Scoring rules

Narrative/explanations

Future TODOs

3.2 — Admin Commands

Create:

docs/ADMIN_COMMANDS.md

List of ALL admin-* commands

How permissions work

What each command does

How to use them

Error handling
