# API: Total MIKROS Contributors

## Feature Overview

Returns the total number of developers, testers, and players registered in the MIKROS ecosystem, showcasing community size and engagement.

---

## Request Details

**Method**: `GET`  
**URL**: `https://api.tatumgames.com/api/gamestats/total-contributors`

---

## Sample Response

```json
{
  "total_contributors": 45823,
  "breakdown": {
    "game_developers": 3420,
    "qa_testers": 1280,
    "beta_players": 8950,
    "community_members": 25173,
    "content_creators": 7000
  },
  "growth_30d": 12.8,
  "timestamp": "2025-10-07T22:00:00Z"
}
```
